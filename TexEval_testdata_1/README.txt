TExEval_testdata_1.0
================================================================================

This package contains the following files:

  - README.txt: a textual description of the TExEval_testdata_1.0 package.
  - chemical.terms: all the terms of a manual gold-standard taxonomy rooted on "chemical";
  - equipment.terms: all the terms of a manual gold-standard taxonomy rooted on "equipment";
  - food.terms: all the terms of a manual gold-standard taxonomy rooted on "food";
  - science.terms: all the terms of a manual gold-standard taxonomy rooted on "science";
  - WN_chemical.terms: all the terms of the WordNet gold-standard taxonomy rooted on "chemical";
  - WN_equipment.terms: all the terms of the WordNet gold-standard taxonomy rooted on "equipment";
  - WN_food.terms: all the terms of the WordNet gold-standard taxonomy rooted on "food";
  - WN_science.terms: all the terms of the WordNet gold-standard taxonomy rooted on "science";


Each file for the provided domain terminologies is tab-separated as follows: 

    term_id <TAB> term

where:
  - term_id is a term identifier (numeric);
  - term is a domain term.
